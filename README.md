\# Customer Segmentation Project



\## Overview

This project uses \*\*Machine Learning (K-Means Clustering)\*\* to segment customers into different groups based on their \*\*Age, Annual Income, and Spending Score\*\*.



The purpose of this project is to help businesses understand customer behavior and improve marketing strategies.



\## Dataset

The project uses the \*\*Mall Customers Dataset\*\* containing:

\- Customer ID

\- Gender

\- Age

\- Annual Income

\- Spending Score



Dataset Link:  

https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python



\## Technologies Used

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Scikit-learn

\- Jupyter Notebook



\## Algorithm Used

\*\*K-Means Clustering\*\* (Unsupervised Machine Learning)



\## Project Workflow

1\. Load Dataset  

2\. Data Preprocessing  

3\. Feature Scaling  

4\. Apply K-Means Clustering  

5\. Visualize Customer Clusters  

6\. Analyze Results
